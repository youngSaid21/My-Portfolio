Exploring Nobel Prize Dynamics: A Correspondence Analysis of Countries and Disciplines

Data Analysis 2024

Description

This project conducts a Correspondence Analysis (CA) on Nobel Prize data, exploring the relationships between countries and disciplines such as mathematics, chemistry, and physics. Utilizing Python, the CA visualizes the distribution of laureates, providing insights into trends in Nobel Prize allocations. The analysis aims to reveal patterns regarding which countries excel in specific fields and how these trends have evolved over time, highlighting the global impact of Nobel-recognized research across different scientific domains.

Tools and Technologies

Pandas Numpy Matplotlib Fanalysis
