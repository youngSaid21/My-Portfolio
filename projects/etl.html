ETL Project: LightAdventureWorks Data Warehouse

Data Analysis 2025

Description


This project involves the creation of a lightweight data warehouse, LightAdventureWorksDW, designed to streamline data analysis and reporting. The ETL (Extract, Transform, Load) process begins with extracting data from an OLTP source, followed by essential transformations to prepare the data for warehousing. The project includes the development of fact tables that support analytical queries and reporting functionality. Utilizing Power BI, users can generate insightful reports and visualizations based on the data housed in LightAdventureWorksDW. This project aims to enhance decision-making capabilities by providing a structured and accessible data environment.
