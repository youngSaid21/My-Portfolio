Gym Management System

Web Development 2024

Description

This project is a user-friendly Gym Management System built with Django to simplify gym operations and improve member experience. It includes features like member registration, attendance tracking, class scheduling. Gym staff can easily manage daily tasks, while members can access their profiles, book classes, and track their fitness progress. The goal is to enhance efficiency and create an engaging environment for fitness enthusiasts.

Tools & Technologies

HTML CSS JavaScript Boostrap UML PostgreSQL Python & Django
